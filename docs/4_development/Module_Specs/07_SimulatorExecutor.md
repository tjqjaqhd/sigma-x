# SimulatorExecutor 모듈 사양

## 1. 모듈 개요
* 모듈명: SimulatorExecutor
* 작성자: SIGMA 팀
* 작성일: 2025-05-22
* 최종 검토자: SIGMA 팀

## 2. 구조 개요
* 포함된 클래스/함수:
  - `SimulatorExecutor`
* 주요 메서드:

* 외부 API 제공 여부: 없음
* 소스 파일 위치: `sigma/core/execution.py`
