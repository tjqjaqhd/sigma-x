## 3. 인터페이스 명세
### 3.1 입력
* 입력 유형: 시세 데이터, 파생 지표
* 형식 및 구조: Pandas DataFrame
* 제약 조건: 모델이 요구하는 피처 스케일 적용
* 예시: `df[["close","volume"]]`

### 3.2 출력
* 출력 유형: 예측 신호 값
* 형식 및 구조: `{signal: float}`
* 예시: `{"signal":0.7}`
* 반환 조건: 모델 추론 완료 시
